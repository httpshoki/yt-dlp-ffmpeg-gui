# yt-dlp and FFmpeg GUI

A simple Python GUI for downloading and converting videos using `yt-dlp` and `FFmpeg`.

## Requirements

- Python 3.x
- yt-dlp (`pip install yt-dlp`)
- FFmpeg (installed and added to PATH)
